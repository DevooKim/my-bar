# My Bar

macOS 메뉴바 아이콘을 숨겨 주는 앱. 구분자(`|`) 왼쪽으로 ⌘드래그한 아이콘은
접을 때 숨겨지고, 메인 아이콘 클릭 또는 ⇧⌘\ 로 펼치고 숨깁니다.

- 특수 권한 불필요 (손쉬운 사용·화면 기록 모두 필요 없음)
- 설정한 시간 뒤 자동 재숨김
- "항상 숨김" 영역 옵션 (⌥+클릭으로만 표시)
- GitHub Releases 기반 자체 업데이트

## 설치

1. [Releases](https://github.com/DevooKim/my-bar/releases/latest)에서 zip을 받아 압축 해제합니다.
2. `My Bar.app`을 응용 프로그램 폴더로 옮기고 실행합니다.
3. 자체 서명 앱이므로 첫 실행 시 시스템 설정 > 개인정보 보호 및 보안에서
   "그대로 열기"로 허용해 주세요.

## 사용법

- 메뉴바 아이콘을 **⌘를 누른 채 드래그**해 `|` 구분자 왼쪽/오른쪽으로 옮깁니다.
- 메인 아이콘(❮/❯) **클릭** 또는 **⇧⌘\\** 로 펼치고 숨깁니다.
- 메인 아이콘 **우클릭**으로 메뉴(설정·업데이트·종료)를 엽니다.
- **⌥+클릭**하면 항상 숨김 영역까지 표시됩니다 (설정에서 활성화).
