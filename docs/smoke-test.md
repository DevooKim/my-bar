# 스모크 테스트

릴리스 전 필수 수동 QA 체크리스트 — `make run` 후 순서대로 확인.

확인 항목 (모두 통과해야 함):
1. 메뉴바 오른쪽에 ❮ 아이콘과 그 왼쪽에 `|` 구분자가 보인다. 첫 실행이면 온보딩 창이 뜨고 펼침 상태다.
2. 온보딩 "시작하기" → 창 닫힘. 이후 재실행 시 온보딩이 다시 뜨지 않는다.
3. ⌘드래그로 아무 아이콘(예: Wi-Fi)을 `|` 왼쪽으로 이동 → 메인 아이콘 클릭 → 해당 아이콘이 사라지고(접힘) 아이콘 방향이 바뀐다. 다시 클릭 → 나타난다.
4. ⇧⌘\ 로도 토글된다.
5. 펼친 뒤 10초 방치 → 자동으로 접힌다.
6. 우클릭 메뉴: 정보/토글/업데이트 확인/설정(⌘,)/종료(⌘Q) 구성 확인. "My Bar 정보" → 정보 창(아이콘·버전·GitHub 링크). "설정..." → 사이드바(일반/정보) 창.
7. 설정 > 일반에서 "항상 숨김 영역 사용" on → 메뉴바에 ⋮ 구분자 등장, ⌥+클릭으로만 그 왼쪽이 보인다.
8. 설정에서 단축키 변경·제거가 동작한다.
9. 앱 종료 후 재실행 → 접힘 상태·영역 구분 유지.

문제 발견 시 이 태스크 안에서 수정 후 다시 확인한다.
